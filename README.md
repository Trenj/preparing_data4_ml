Лабораторные работы по обработке табличных данных (Python, Pandas).

## Структура
- `lab1/` — базовые операции с CSV и визуализацией:
  - `main.ipynb` — чтение данных JCMB (2006/2014), установка индекса по дате, выбор столбцов, графики `plot`, создание поднабора `weather_100_rows.csv`, подсчёт уникальных значений и топ‑10 по скорости ветра.
  - `JCMB_2006_Jun.csv`, `JCMB_2014_Aug.csv` — исходные метеоданные.
- `lab2/` — анализ датасета болезни Паркинсона:
  - `materials/ПЗ_2_ РАБОТА С ТАБЛИЧНЫМИ ДАННЫМИ.py` — шаблон с заданиями (интеграция данных, очистка, корреляция, отбор признаков, визуализация).
  - `materials/ПЗ_2_ РАБОТА С ТАБЛИЧНЫМИ ДАННЫМИ (для преп).py` — пример решения тех же шагов.
  - `parkinson_disease.csv` — датасет 756×755 для заданий.

## Как запускать
1. Установить зависимости: `pip install pandas numpy matplotlib seaborn scikit-learn`.
2. Открыть соответствующий `.ipynb` или `.py` в IDE/Colab и выполнять ячейки/блоки по порядку.

## Что сдавать
- Для `lab1`: ноутбук `lab1/main.ipynb` (и при необходимости сгенерированный `weather_100_rows.csv`).
- Для `lab2`: заполненный `materials/ПЗ_2_ РАБОТА С ТАБЛИЧНЫМИ ДАННЫМИ.py` (либо ноутбук-эквивалент); файл «для преп» — только как пример.
